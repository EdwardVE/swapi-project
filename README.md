# SWAPI Project

## Descripción
Esta aplicación Angular permite visualizar planetas, sus residentes y los vehículos de estos, utilizando la API de Star Wars (https://swapi.dev).

## Instalación
1. Clonar el repositorio.
   ```bash
   git clone [<repository-url>](https://github.com/EdwardVE/swapi-project.git)
   cd swapi-project
Instalar las dependencias.
npm install
Ejecutar la aplicación.
npm start
Navegar a http://localhost:3000.
